Research Projects

Multi-Sensor Fusion for Controlling a Soft Pneumatic Hand Exoskeleton

Real-Time Machine Learning Application & Soft Robotics

Designed and optimized Ecoflex-based soft pneumatic actuators using FEA, achieving a high bending angle of 288.79° with external thread reinforcement.

  • Developed a real-time gesture recognition system using an armband with ADXL335 and IMU sensors to control the exoskeleton
  • Implemented and compared ML models (XGBoost, Random Forest, SVM), with Random Forest achieving 93.01% accuracy for reliable actuator control
  • Designed and optimized Ecoflex-based soft pneumatic actuators using FEA
  • Achieved high bending angle of 288.79° with external thread reinforcement

Biomimetic Fish Robot With Fluidic Actuation

Soft Robotics

Designed a fish robot incorporating fluidic actuation for biomimetic underwater movement. Developed and tested soft actuators with varying numbers of chambers to study deformation behavior.

  • Conducted comparative analysis of chamber configurations
  • Optimized actuator design for efficient underwater propulsion
  • Developed fluid control systems for precise movement
  • Tested biomimetic motion patterns in aquatic environments

Interactive GRE Vocabulary with Nao Robot

Human-Robot Interaction

Developed an interactive vocabulary learning system using the NAO robot to assist with GRE preparation. Programmed the robot to teach word meanings, explain usage, and provide sentence examples.

  • Integrated comprehensive GRE word list database
  • Designed interactive learning sessions with NAO robot
  • Implemented natural language processing for Q&A
  • Created user progress tracking system

Pneumonia Detection from Chest X-rays

Medical Image Analysis & Deep Learning

Developed a CNN-based model to classify chest X-rays as normal or pneumonia-infected. Achieved 92% accuracy using transfer learning with data augmentation techniques.

  • Implemented Grad-CAM visualization to highlight regions influencing the diagnosis
  • Used transfer learning with data augmentation for improved accuracy
  • Developed CNN-based classification model
